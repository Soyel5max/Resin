## 🔧 Features Explained

Resin offers practical tools to manage your proxy network:

- **Proxy Pool Management:** Combine many proxies into one network. Resin balances the load and avoids downtime.
- **Sticky Sessions:** Resin keeps the same proxy for your sessions, improving reliability for your apps.
- **Status Monitoring:** See which proxies are active and how fast they are inside the dashboard.
- **Automatic Updates:** Resin can update proxy lists automatically when you set subscriptions.
- **Error Handling:** Resin detects bad proxies and removes them from the pool.

## 🛠️ Basic Troubleshooting

If Resin does not run as expected:

- Make sure you downloaded the Windows version.
- Check if your antivirus blocks the app.
- Restart your computer after installing Resin.
- Verify internet connection is active.
- Confirm proxy subscriptions are valid and formatted correctly.
- Close other programs that might use the same proxies.
